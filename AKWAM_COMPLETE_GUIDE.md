```typescript
// مثال: إضافة فيلم جديد
await contentManager.addMovie({
  title: "الفيلم الجديد",
  year: "2024",
  genre: ["أكشن", "دراما"],
  quality: "4K",
  plot: "وصف الفيلم...",
  poster: "/images/poster.jpg",
  featured: true

```

### 🔄 **تحديث المحتوى**
```typescript
// تحديث فيلم موجود

  rating: "8.5",

  ]
})
```

### 🗑️ **حذف المحتوى**
```typescript
// حذف فيلم

```
