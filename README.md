## 🌟 **نظرة عامة**

موقع بث متكامل يحاكي **ak.sv** بدقة مع إضافة نظام مراقبة وصيانة ذكي متطور. تم تطويره باستخدام أحدث التقنيات لضمان الأداء العالي والموثوقية.

## 🚀 **التقنيات المستخدمة**

### Frontend:
- **Next.js 14** - App Router مع React 18
- **TypeScript** - للأمان والموثوقية
- **Tailwind CSS** - تصميم سريع ومرن
- **Radix UI** - مكونات واجهة مستخدم عالية الجودة
- **Framer Motion** - رسوم متحركة سلسة
- **Zustand** - إدارة الحالة البسيطة

### Backend:
- **Next.js API Routes** - خادم متكامل
- **Prisma ORM** - إدارة قاعدة البيانات
- **SQLite** - قاعدة بيانات سريعة ومحمولة
- **NextAuth.js** - نظام مصادقة آمن
- **Winston** - تسجيل متقدم
- **Node-cron** - جدولة المهام

### Monitoring & DevOps:
- **Custom Health Checker** - نظام مراقبة مخصص
- **Nodemailer** - إرسال الإشعارات
- **Axios** - طلبات HTTP موثوقة
- **Systeminformation** - معلومات النظام
- **Daily Log Rotation** - إدارة ملفات السجل

## 📦 **التثبيت والتشغيل**

### المتطلبات:
- Node.js 18+ 
- npm أو yarn
- Git

### خطوات التثبيت:

```bash
# 1. استنساخ المشروع
git clone https://github.com/your-username/ak-sv-streaming.git
cd ak-sv-streaming

# 2. تثبيت التبعيات
npm install

# 3. إعداد قاعدة البيانات
cp .env.example .env
npm run db:generate
npm run db:push

# 4. تشغيل المشروع
npm run dev

# 5. تشغيل نظام المراقبة (في terminal منفصل)
npm run monitor
```

### ⚙️ **الإعداد**

#### متغيرات البيئة (.env):

```env
# Database
DATABASE_URL="file:./dev.db"

# NextAuth.js
NEXTAUTH_SECRET="your-secret-key-here"
NEXTAUTH_URL="http://localhost:3000"

# Email للإشعارات
SMTP_HOST="smtp.gmail.com"
SMTP_PORT="587"
SMTP_USER="your-email@gmail.com"
SMTP_PASSWORD="your-app-password"
SMTP_FROM="noreply@ak-sv.com"

# إعدادات الإدارة
ADMIN_EMAIL="admin@ak-sv.com"

# مراقبة
MONITORING_ENABLED="true"
HEALTH_CHECK_INTERVAL="300000"
LOG_LEVEL="info"
```

## 🔧 **التخصيص والتطوير**

### إضافة صفحة جديدة للمراقبة:
```typescript
// في أي مكان في التطبيق
import { HealthChecker } from '@/lib/monitoring/health-checker'

const healthChecker = HealthChecker.getInstance()
await healthChecker.addPageMonitor(
  'https://example.com', 
  'صفحة المثال'
)
```

### إضافة خادم جديد للمراقبة:
```typescript
await healthChecker.addServerMonitor(
  'server-id',
  'اسم الخادم',
  'https://server.example.com'
)
```

### تسجيل حدث مخصص:
```typescript
import { logger } from '@/lib/monitoring/logger'

logger.info('عملية مكتملة', {
  userId: 'user-123',
  metadata: { action: 'upload', fileSize: 1024 }
})
```
