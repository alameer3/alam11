# 🎬 أكوام - موقع الأفلام والمسلسلات

## 📖 نظرة عامة

موقع أكوام هو منصة شاملة لمشاهدة وتحميل الأفلام والمسلسلات والبرامج التلفزيونية. تم تطويره باستخدام أحدث التقنيات لتوفير تجربة مستخدم استثنائية.

## ✨ الميزات الرئيسية

### 🎯 للمستخدمين
- **مشاهدة متقدمة**: مشغل فيديو متطور مع دعم الترجمة
- **بحث ذكي**: نظام بحث متقدم مع فلاتر شاملة
- **تصنيفات متنوعة**: أفلام، مسلسلات، برامج، ومحتوى مختلط
- **واجهة عربية**: دعم كامل للغة العربية والاتجاه RTL
- **تجربة تفاعلية**: تقييمات، تعليقات، ومفضلة
- **تصميم متجاوب**: يعمل على جميع الأجهزة

### 🛡️ للإدارة
- **لوحة تحكم شاملة**: إدارة كاملة للمحتوى والمستخدمين
- **إدارة الملفات**: رفع وتنظيم الفيديوهات والصور
- **نظام المستخدمين**: إدارة الحسابات والصلاحيات
- **إحصائيات مفصلة**: تقارير شاملة عن الاستخدام
- **إدارة الخوادم**: إدارة مصادر المحتوى المختلفة

## 🚀 التقنيات المستخدمة

### Frontend
- **Next.js 14** - إطار عمل React متطور
- **TypeScript** - لكتابة كود آمن ومنظم
- **Tailwind CSS** - تصميم سريع ومرن
- **Radix UI** - مكونات UI متقدمة
- **Framer Motion** - رسوم متحركة سلسة

### Backend
- **Next.js API Routes** - واجهات برمجة التطبيقات
- **Prisma** - ORM متطور لقواعد البيانات
- **NextAuth.js** - نظام مصادقة آمن
- **SQLite** - قاعدة بيانات سريعة (قابلة للتطوير)

### المكتبات المساعدة
- **Lucide React** - أيقونات حديثة
- **Sharp** - معالجة الصور
- **Swiper** - عروض تقديمية تفاعلية
- **Recharts** - رسوم بيانية متطورة

## 📦 التثبيت والإعداد

### 1. الإعداد السريع (موصى به)
```bash
# استنساخ المشروع
git clone https://github.com/your-username/𝐘𝐄𝐌𝐄𝐍_𝐅𝐋𝐈𝐗-clone.git
cd 𝐘𝐄𝐌𝐄𝐍_𝐅𝐋𝐈𝐗-clone

# تشغيل الإعداد التلقائي
node scripts/setup.js
```

### 2. الإعداد اليدوي
```bash
# تثبيت التبعيات
npm install

# إعداد قاعدة البيانات
npx prisma generate
npx prisma db push

# إضافة بيانات تجريبية (اختياري)
npm run seed

# تشغيل المشروع
npm run dev
```

## 🎮 الاستخدام

### للمطورين
```bash
# تشغيل الخادم المطور
npm run dev

# بناء المشروع للإنتاج
npm run build

# تشغيل النسخة المبنية
npm start

# فحص الأكواد
npm run lint
```

### للمستخدمين
1. **زيارة الموقع**: افتح `http://localhost:3000`
2. **التصفح**: استخدم القوائم للانتقال بين الأقسام
3. **البحث**: استخدم شريط البحث للعثور على المحتوى
4. **المشاهدة**: انقر على أي محتوى للمشاهدة

### للإدارة
1. **تسجيل الدخول**: `http://localhost:3000/auth/signin`
2. **لوحة الإدارة**: `http://localhost:3000/admin`
3. **إضافة محتوى**: استخدم لوحة الإدارة لإضافة أفلام ومسلسلات
4. **إدارة المستخدمين**: عرض وإدارة حسابات المستخدمين
