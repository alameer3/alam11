## 📋 **دليل التشغيل**

### **1. متطلبات التشغيل:**
```bash
Node.js 18+ 
npm 8+
Git
```

### **2. تنزيل وتشغيل المشروع:**
```bash
# استنساخ المشروع
git clone <repository-url>
cd ak-sv-clone

# تثبيت التبعيات
npm install

# إعداد قاعدة البيانات
npx prisma generate
npx prisma db push

# تشغيل التطبيق
npm run dev
```

- 🌐 **الموقع الرئيسي:** `http://localhost:3000`
- ⚙️ **لوحة التحكم:** `http://localhost:3000/admin`
